## 🚀 วิธีการรันโปรเจกต์

เนื่องจากไฟล์ทั้งหมดเป็นสกุล `.html` ที่มี JavaScript ฝังอยู่ คุณสามารถดูผลลัพธ์ได้ง่ายๆ บน Web Browser:

**Clone Repository** ลงมาที่เครื่องคอมพิวเตอร์ของคุณ:
   ```bash
   git clone [https://github.com/gatunyus/Frontend_Day1_JS.git](https://github.com/gatunyus/Frontend_Day1_JS.git)
   ```
**การเปิดดูผลลัพธ์และทดสอบโค้ด:**
   - เปิดไฟล์ `.html` ที่ต้องการด้วย Web Browser (เช่น Google Chrome)
   - **สำคัญ:** ผลลัพธ์ของ JavaScript หลายๆ ตัวมักจะแสดงผลใน **Console** - ให้คลิกขวาที่หน้าเว็บ เลือก **Inspect (ตรวจสอบ)** หรือกดปุ่ม `F12` แล้วไปที่แท็บ **Console** เพื่อดูผลลัพธ์การทำงานของสคริปต์ หรือข้อความที่ใช้ `console.log()`
**การเปิดด้วย Live Server (แนะนำ):**
   - เปิดโฟลเดอร์โปรเจกต์นี้ด้วย Text Editor เช่น **VS Code** - หากติดตั้ง Extension `Live Server` ไว้ ให้คลิกขวาที่ไฟล์แล้วเลือก `Open with Live Server` เพื่อให้หน้าเว็บรีเฟรชอัตโนมัติเมื่อมีการแก้ไขโค้ด
